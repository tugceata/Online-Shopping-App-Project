# 🛍️ Online Alışveriş Sitesi (Java + Swing)

Bu proje, Java dili kullanılarak geliştirilen ve Swing kütüphanesi ile arayüzü oluşturulan bir online alışveriş sitesi uygulamasıdır. Uygulama hem kullanıcı hem de yönetici rolleri içermektedir ve tamamen **nesne tabanlı programlama (OOP)** prensipleri ile geliştirilmiştir.

## 🎯 Amaç

Projenin amacı, farklı kategorilerdeki ürünlerin (teknoloji, moda, kitap, vb.) yönetilebildiği ve satın alınabildiği bir e-ticaret sisteminin masaüstü ortamda simülasyonunu gerçekleştirmektir.

## 👥 Kullanıcı Rolleri

### 👤 Müşteri
- Kayıt olabilir, giriş yapabilir
- Ürünleri kategori bazlı görüntüleyebilir
- Ürünleri sepete ekleyip çıkarabilir
- Sepet içeriğini düzenleyebilir

### 🛠️ Yönetici
- Ürün ekleyebilir, düzenleyebilir, silebilir
- Kullanıcıları görüntüleyebilir ve yönetebilir
- Satın alınan ürünleri raporlayabilir

## 🧰 Geliştirme Ortamı

- Java 8+
- Eclipse IDE (bin klasörü içeriyor)
- Swing GUI
- OOP prensipleri


## 📁 Veri Yapısı

- `kullanicilar.txt` – Kayıtlı kullanıcı bilgileri
- `cart.txt` – Sepet verileri
- `*.txt` – Her kategoriye özel ürün dosyaları

## 🛠️ Kurulum ve Çalıştırma

1. Java 8+ kurulu olduğundan emin olun
2. Projeyi bir Java IDE (örneğin Eclipse veya IntelliJ) ile açın
3. `HomePage` sınıfını çalıştırarak uygulamayı başlatın
